About Us

=

Here we propose and a Sentiment Analysis for Product Rating system that detects hidden sentiments in comments and rates the product accordingly. The system uses sentiment analysis methodology in order to achieve the desired functionality. This project is an E-Commerce web application where the user will view the product and product features and the system will analyze the reviews of various users and will rank the product. The system would initially scrape the reviews of the user’s desired product. Each of the scrapped reviews is processed by the sentiment classifier to determine the polarity. The reviews are classified as positive or negative. The sentiment classifier is trained through a training set to differentiate negative and positive comments. The ratio of the negative and positive comments is computed and the analysis is graphically visualized.


The user interface facilitates the user to input an ASIN of any product, whose reviews are to be classified as bearing positive or negative polarity from amazon.com. Amazon Standard Identification Numbers (ASINs) are unique blocks of 10 letters and/or numbers that identify items. You can find the ASIN on the item's product information page at Amazon.com. For books, the ASIN is the same as the ISBN number, but for all other products, a new ASIN is created when the item is uploaded to Amazon’s catalog.
